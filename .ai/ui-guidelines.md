UI Design Rules:

Design Identity:
- Minimal
- Futuristic
- Airy
- Calm
- Professional SaaS workstation

Theme:
- Use CSS variables only
- bg-main
- bg-panel
- textPrimary
- textMuted
- accentAI
- border-soft

Layout:
- Use gap-6 spacing
- Use p-6 padding
- Rounded-2xl panels
- Avoid heavy shadows
- Avoid gradients
- Avoid neon colors

Sidebar:
- Collapsible
- Icon-only when collapsed
- Tooltip on hover when collapsed
- Smooth width transition
